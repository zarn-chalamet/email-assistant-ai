📬 AI-Powered Email Auto Reply Generator & Research Assistant


A productivity-focused project consisting of two Google Chrome extensions integrated with a Spring Boot backend and Google Gemini API.

Email Auto Reply Generator: Automatically drafts professional, context-aware replies in Gmail.

Research Assistant: Summarizes any selected text from the browser.

_____________________________________________________


🚀 Features

Email Auto Reply Generator

-  Adds an AI Reply button to Gmail's compose interface.

-  Automatically extracts the latest incoming email content.

-  Sends the context to the backend for Google Gemini API processing.

-  Generates a professional, ready-to-send reply.

Research Assistant

-  Chrome extension sidebar tool.

-  Users can highlight text on any webpage.

-  Generates clear, concise summaries via Google Gemini API.


🛠️ Tech Stack

Frontend: Google Chrome Extension APIs, JavaScript, HTML, CSS

Backend: Spring Boot (Java)

AI Integration: Google Gemini API

Communication: REST APIs (JSON)

_____________________________________________________


🔄 How It Works

Email Auto Reply Flow

1.  User clicks AI Reply in Gmail.

2.  Extension extracts last email content.

3.  Sends it to backend.

4.  Backend calls Google Gemini API.

5.  AI-generated reply is returned and inserted into Gmail draft.

Research Assistant Flow

1.  User selects text on any webpage.

2.  Clicks Summarize in extension sidebar.

3.  Sends selected text to backend.

4.  Backend calls Google Gemini API.

5.  Summary is displayed in sidebar.
